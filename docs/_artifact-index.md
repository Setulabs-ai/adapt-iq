## What AdaptIQ Is (inferred)

A Shopify app ("AdaptIQ") that installs via OAuth, injects a storefront script (`public/widget.js`) via a Shopify ScriptTag, and serves AI-generated product recommendations, cross-sell bundles, cart upsells, adaptive hero copy, and semantic search — all backed by OpenAI (`gpt-4o-mini`) and a Supabase Postgres database. Merchants manage the app through a Next.js dashboard (`/dashboard`) gated by a JWT session cookie.

- `src/app/api/recommendations/route.ts`
